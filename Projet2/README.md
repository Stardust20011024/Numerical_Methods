## Projet 2：高阶有限元方法求解二阶非齐次线性常微分方程

本项目研究**P2 有限元方法**在二阶非齐次线性常微分方程中的应用。重点在于有限元方法的理论构造、数值实现以及误差分析。

本报告共分为三部分：

### 第一部分：精确解的推导

首先对所研究的线性常微分方程系统给出**解析精确解**的计算过程。在推导过程中需特别注意**非齐次边界条件**的处理。

### 第二部分：P2 有限元数值方法

本部分介绍采用**P2 有限元方法**对问题进行数值离散，具体包括：

- **变分形式的建立**：  
  推导问题的变分形式，对应项目中练习 1 的问题 (a)。
- **矩阵形式的构造与求解**：  
  给出有限元离散后的矩阵表达形式，并用于数值求解，对应练习 1 的问题 (b) 和 (c)。

### 第三部分：误差分析

对 P2 有限元数值解的误差进行系统分析，分别在不同范数下研究其收敛性质，包括：

- **$L^2$ 范数下的误差分析**
- **半范数（semi-norm）下的误差分析**

通过误差计算验证高阶有限元方法在精度上的优势。
